# DriverLinkPLW
# Asisten Pintar untuk Driver Maximl.

Latar Belakang:
Driver ojol sering harus WA pelanggan sambil nyetir motor. Harus copy nomor, buka WA, paste, ketik pesan — ribet dan bahaya. Belum lagi harus hitung sendiri berapa uang bersih yang didapat setelah potongan aplikasi. OjolLink Bot hadir buat nyeleseiin semua itu dalam satu alur simpel.

Fitur:
- Kirim nomor → dapat link WA langsung
- Pesan "Hallo dari Maxim" sudah otomatis terisi
- Support semua format nomor HP (08xx, +62xx, 62xx)
- Setelah dapat link, bot langsung tanya tarif
- Kalkulasi otomatis alokasi tarif (komisi 13%, bensin 10%, saldo 10%, kantong 80%)

Teknologi:
- Bahasa: Python 3.13
- Library: python-telegram-bot v21.6
- Bot API: Telegram Bot API & WhatsApp wa.me
- Version control: GitHub
- Deploy: Railway (cloud gratis)

ENGLISH
Background:
Every driver knows the struggle — you're riding, got a new order, and suddenly you gotta open WhatsApp, copy the number, paste it, type "Hallo aku dari Maxim", all while handling the bike. Risky, slow, and straight up annoying. On top of that, calculating how much you actually pocket after app cuts? Total headache. OjolLink Bot kills all that noise in one smooth flow.

Features:
- Drop a number → get a WA link instantly 🔗
- "Hallo dari Maxim" message auto-filled, no typing needed ✍️
- Handles any phone format (08xx, +62xx, 62xx) no cap 📱
- Bot immediately asks for the fare right after sending the link 💬
- Auto-calculates fare breakdown (13% app cut, 10% fuel savings, 10% balance savings, 80% clean cash) 💰

Tech Stack:
- Language: Python 3.13
- Library: python-telegram-bot v21.6
- Bot API: Telegram Bot API & WhatsApp wa.me
- Version Control: GitHub
- Deploy: Railway (free cloud)
